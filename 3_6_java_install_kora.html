জাভা ইন্সটল করা


জাভা ইন্সটল করা বলতে আসলে আমরা JDK(Java Development Kit) ইন্সটল করবো। জাভাতে প্রোগ্রামিং করার জন্য আমাদের আগে JDK ইন্সটল করতে হবে।

প্রথমে একটু দেখে এসো তোমার পিসিতে কত বিট এর উইন্ডোজ ইন্সটল করা আছে।






এটি ৩২ বিট এর উইন্ডোজ। এখানে ৬৪ বিট ও লেখা থাকতে পারে। এখানে যত বিট লেখা থাকবে তোমাকে তত বিটের JDK ডাউনলোড করতে হবে।

Variable name এর জায়গায় লেখো PATH আর Variable value এর জায়গায় C:\Program Files\Java\jdk1.8.0_101\bin লিখে দাও। খেয়াল রেখো যে নতুন PATH হলে C:\ এর সামনে কিন্তু সেমিকোলন(;) থাকবে না।
