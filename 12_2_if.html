If


উপরের যে উদাহরণটি আমরা এতক্ষণ দেখলাম সেটাই মূলত if এর উদাহরণ। এখানে একটি কন্ডিশন চেক করতে বলা হয়। কন্ডিশনের ফলাফল যদি সত্য বা true হয় তাহলে প্রোগ্রাম একটি বা একাধিক, সোজা কথায় তাকে যতগুলো কাজ করতে বলা হবে ততগুলো কাজ করবে।

উপরের উদাহরণটিই আমরা একটি প্রোগ্রামে লিখি:




আউটপুট:




If এর সিনট্যাক্সটি খুব সিম্পল:

if(condition){

//statement

}

এখানে condition এর জায়গায় তুমি যে কোন কন্ডিশন দিতে পারো। একাধিক বা মাল্টিপল কন্ডিশনও দিতে পারো।

আর //statement হলো তোমার দেয়া কাজ। তুমি এখানে যতগুলো কাজ করতে দেবে প্রোগ্রাম ততগুলো কাজই করবে।

আর উল্লেখ্য যে condition এর রেজাল্ট শুধুমাত্র true হলেই কিন্তু //statement এর কাজ সম্পন্ন হবে। false হলে প্রোগ্রাম কোন কিছুই করবে না।

অর্থাৎ একটি if এ প্রোগ্রাম আগে দেখে কন্ডিশন true কি না। যদি true হয় তাহলে //statement এক্সিকিউট করে আর true না হলে কিছুই করে না। যেমন দেখো:
