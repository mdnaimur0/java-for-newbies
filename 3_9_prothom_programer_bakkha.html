প্রথম প্রোগ্রামের ব্যাখ্যা


এই প্রোগ্রামটিতে একট ক্লাশ(class) আছে।

public class MyFirstJavaProgram{

}

এটুকু হচ্ছে একটি ক্লাশ(class)

ক্লাশের মধ্যে আমরা একটি মেইন(main) মেথড(method) রেখেছি।

public class MyFirstJavaProgram{

public static void main(String args[]){

}

}

আর মেইন মেথডের মধ্যে এক লাইনের একটি কোড লিখেছি।

System.out.println("This is my first Java Program");

জাভাতে এটি ব্যবহৃত হয় কোন লেখাকে স্ক্রীণে দেখানোর জন্য।

System.out.println();

এই কোডের () এর মধ্যে ডাবল কোটেশন বা “” এর ভেতরে যা লিখে দেবে জাভা সেটিকেই স্ক্রীণে দেখাবে।

তাহলে শেষমেষ যে কোডটি দাঁড়ালো তা হলো:

১. একটা ক্লাশ(class)

২. তার ভেতরে একটা মেইন(main) মেথড(method)

৩. তার অভ্যন্তরে একটি লাইন যা স্ক্রীণে কোন লেখাকে দেখায় যখন প্রোগ্রামটি রান করানো হয়।

 

নোট: এখানে main কি method কি class কি কিছুই তোমার বোঝার প্রয়োজন নেই। একে একে আমরা সবই ব্যাখ্যা করবো । তুমি শুধু নিশ্চিত করো যে তোমার প্রথম প্রোগ্রামটি রান করেছে কি না। তোমরা যেহেতু নতুন শুরু করছো তাই JDK ইন্সটল করা থেকে শুরু করে এই ষ্টেপ পর্যন্ত একটু আকটু সমস্যা ফেস করতে হতে পারো। কিন্তু চিন্তিত হইও না। কয়েকবার চেষ্টা করলেই পেরে যাবে।
